# DDoS-DoS-attack-simulation
Project in 5G and IoT advanced course. Building a simulation of an DDoS/DoS attack with two raspberry pi using a client server architecture.

## Create a New Feature or Make Other Changes:

- Create a new branch
- Make changes and commit
- Push the branch to origin
- Create a pull request


## How to start

1. Run server in Windows PowerShell terminal
2. When the server is up, start running the clinet code (raspberry pi)
3. The output should be visble in the terminal

## Command to run the server

```bash
python server_PC.py
```
